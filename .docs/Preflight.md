# Preflight Considerations

## Contenders

- Miniflux / https://github.com/miniflux/v2 - web UI, very basic layout, clunky
- Fluentflame Reader / https://github.com/FluentFlame/fluentflame-reader - Electron, no release yet, untested
- NewsBlur / https://github.com/samuelclay/NewsBlur - web UI, nice layouts??? (examples - https://blog.newsblur.com/), self-hosted, untested
- Tiny Tiny RSS / https://github.com/tt-rss/tt-rss - web UI, untested

## Discounted

- RSS Guard / https://github.com/martinrotter/rssguard - dated UI, busy, not good UX
- FreshRSS / https://github.com/FreshRSS/FreshRSS - web UI, clunky, dated
- FeedDesk / https://github.com/torum/FeedDesk - desktop, nice UI but missing too many features, C#
- yarr / https://github.com/nkanaev/yarr - web UI, basic + lacking configuration / settings
